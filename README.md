# DevTier

GitHub 잔디(contribution) 데이터로 한국 개발자 전투력을 측정하고 티어를 부여하는 서비스.
백준 온라인 저지 종료 이후 개발자가 실력을 증명할 새로운 지표.

**🌐 [https://devtier-brown.vercel.app](https://devtier-brown.vercel.app)**

---

## 사용 방법

1. 사이트에서 GitHub 아이디를 입력한다.
2. 잔디 데이터를 분석해 전투력 점수와 티어를 계산한다.
3. 결과 페이지에서 뱃지 코드를 복사해 GitHub 프로필 README에 붙여넣는다.

```markdown
![DevTier](https://devtier-brown.vercel.app/api/badge/깃헙아이디)
```

점수가 바뀌면 뱃지도 자동으로 업데이트된다.

---

## 전투력 점수 계산

GitHub 최근 1년 잔디 데이터를 기반으로 아래 공식으로 계산합니다.

```
전투력 = 총 잔디 수 × 1
       + 현재 연속 스트릭 × 3
       + 최대 스트릭 × 2
       + 잔디 밀도(%) × 100
       + 피크 강도 × 0.5
```

| 지표 | 설명 | 가중치 |
| --- | --- | --- |
| 총 잔디 수 | 최근 1년 contribution 합산 | ×1 |
| 현재 연속 스트릭 | 오늘 기준 연속 커밋 일수 | ×3 |
| 최대 스트릭 | 역대 최장 연속 커밋 기록 | ×2 |
| 잔디 밀도 | 365일 중 커밋한 날 비율 (0~1) | ×100 |
| 피크 강도 | 하루 최대 커밋 수 | ×0.5 |

> 꾸준함(스트릭·밀도)에 높은 가중치를 둬, 단기 몰아치기보다 지속적인 활동이 유리합니다.

## 티어 구조

| 티어 | 기준 | 색상 |
| --- | --- | --- |
| 챌린저 | 한국 상위 100명 (절대 인원 고정) | `#FF4655` |
| 다이아 1~4 | 상위 1% ~ 5% | `#56C8D8` |
| 플래티넘 1~4 | 상위 5% ~ 15% | `#5AC9A6` |
| 골드 1~4 | 상위 15% ~ 30% | `#FFD700` |
| 실버 1~4 | 상위 30% ~ 50% | `#C0C0C0` |
| 브론즈 1~4 | 상위 50% ~ 100% | `#CD7F32` |

티어 내 숫자가 낮을수록 높은 등급 (1이 최고).

**백분위 산정 기준:** GitHub location을 한국·서울·Korea로 설정한 유저 중 수집된 인원 기준. 챌린저는 백분위와 무관하게 전체 점수 상위 100명으로 고정됩니다.

**티어 내 숫자(1~4) 결정:** 해당 티어 구간을 4등분해 상위 25%씩 나눕니다. 예를 들어 골드(상위 15~30%) 구간 안에서도 상위 25%면 골드 1, 하위 25%면 골드 4.
