## 프리미엄 / 결제 시스템

> 뱃지가 충분히 퍼진 이후 (목표: GitHub 뱃지 1만 개) 진행.

- [ ] GitHub OAuth 로그인 (Supabase Auth)
- [ ] `users` 테이블 `is_pro BOOLEAN DEFAULT FALSE` 컬럼 추가 마이그레이션

### 결제

- [ ] 결제 연동 (Paddle 또는 Stripe, ₩990~1,490/월)

## 출신학교 인증 시스템

> 학교 이메일 또는 학생증으로 재학/졸업 여부를 인증하고 프로필에 표시.

### 인증 방식 (미확정, 논의 필요)
- 대학교 이메일 도메인 인증 (e.g. `@snu.ac.kr`, `@kaist.ac.kr`)
- 학교 인증 후 결과 페이지·랭킹에 뱃지 노출

### 태스크

- [ ] 학교 도메인 목록 테이블 (`schools`) 설계
- [ ] `users` 테이블 `school TEXT` 컬럼 추가
- [ ] 학교 이메일 인증 플로우 (이메일 발송 → 토큰 검증)
- [ ] 결과 페이지에 학교 뱃지 노출 UI
- [ ] 랭킹 테이블 학교 필터 기능
